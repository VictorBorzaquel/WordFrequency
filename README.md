# Word Frequency

## React Native App

Esse aplicativo você consegue ver em  lista e gráfico a frequência das palavras em um texto, ou seja, calcula quantas palavras repetidas tem no texto.


Para fazer o gráfico utilizei o Victory Pie.

## Tecnologias

- React Native
- Styled Components
- Typescript
- Victory Pie
